## Español

Formulario web para el registro y cálculo de métricas de software de la Dirección de Desarrollo de Capacidades (DDC) de la Secretaría de la Superintendencia Nacional de Fiscalización (SSNF) de Panamá.

### Contenido

- `formulario_ddc_ssnf_v2.html` — Formulario web en HTML con validación y cálculos
- `Metricas_Software_DDC_SSNF.docx` — Documento de métricas en Word
- `Metricas_Software_DDC_SSNF.pdf` — Documento de métricas en PDF
- `diagrama_casos_uso_ddc.png` — Diagrama de casos de uso

## English

Web form for registering and calculating software metrics from the Capacity Development Directorate (DDC) of the National Supervision Secretariat (SSNF) of Panama.

### Contents

- `formulario_ddc_ssnf_v2.html` — HTML web form with validation and calculations
- `Metricas_Software_DDC_SSNF.docx` — Metrics document in Word format
- `Metricas_Software_DDC_SSNF.pdf` — Metrics document in PDF format
- `diagrama_casos_uso_ddc.png` — Use case diagram
